## Getting Started

1. Clone the project.
2. Go to folder directory.
3. Run following command:

   ```
   npm install react
   npm install react-dom
   npm install react-scripts
   npm install react-tooltip

   ```

4. Run command: `npm start` .
5. Happy Learning.
